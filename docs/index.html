Functional programming

Functions

Functions are first-class

Pure functions

Now with functions!


        # Applications are left associative
       '1'      ( 位x.x )( 位y.y ) z
       '2'  ( ( 位x.x )( 位y.y ) ) z # rewrite as
       '3'       [ x := ( 位y.y ) ] # apply from left to right
       '4'              ( 位y.y ) z
       '5'              [ y := z ]
       '6'                       z # reduced to identity
       '7'                         # "z" is a free variable
